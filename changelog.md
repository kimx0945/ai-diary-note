# 변경 이력 (Changelog)

이 파일은 'AI 감성 일기' 웹앱 프로젝트의 모든 변경 사항을 시간 순으로 기록합니다.

## [2026-04-24 15:20 KST]
### ✨ 새로 추가된 기능 (Added)
- **실시간 채팅 기능 추가**:
  - Supabase Realtime을 활용하여 사용자 간 실시간 메시지 전송 및 수신 기능을 구현했습니다.
  - 글래스모피즘 스타일의 채팅 UI와 자동 스크롤 기능을 추가했습니다.
  - 본인이 보낸 메시지와 다른 사람이 보낸 메시지를 시각적으로 구분했습니다.

## [2026-04-24 14:42 KST]
### ✨ 새로 추가된 기능 (Added)
- **사용자별 데이터 격리 및 보안 강화**:
  - 프론트엔드(`script.js`)에서 API 호출 시 Supabase `access_token`을 `Authorization` 헤더에 포함하도록 수정했습니다.
  - 서버리스 백엔드(`api/analyze.js`, `api/history.js`)에서 토큰을 검증하여 실제 사용자의 ID를 추출하는 보안 로직을 구현했습니다.
  - Redis 저장 키 구조를 `user:[userId]:diary-*`로 변경하여, 각 사용자가 자신의 일기 데이터만 저장하고 조회할 수 있도록 완벽하게 격리했습니다.
  - 이제 다른 계정으로 로그인하면 해당 사용자의 기록만 나타납니다.
## [2026-04-24 13:48 KST]
### ✨ 새로 추가된 기능 (Added)
- **Supabase 인증을 활용한 로그인 화면 구현**:
  - 앱 접속 시 로그인 여부를 확인하여, 비로그인 사용자에게는 아름다운 글래스모피즘 스타일의 로그인 화면을 제공합니다.
  - 이메일/비밀번호 기반의 '로그인' 및 '회원가입' 기능을 추가했습니다.
  - 'Google로 로그인' 버튼을 추가하여 소셜 로그인을 지원합니다.
  - 로그인 성공 시 부드럽게 기존의 일기장 화면으로 전환되며, 상단 우측에 '로그아웃' 버튼이 표시됩니다.
  - Vite 환경에서 Vercel의 Supabase 환경 변수를 안전하게 클라이언트로 전달하기 위해 `vite.config.js` 설정을 추가했습니다.
## [2026-04-24 13:42 KST]
### ✨ 새로 추가된 기능 (Added)
- **Supabase 연동 기초 설정**:
  - `@supabase/supabase-js` 라이브러리를 설치하고 프로젝트에 통합했습니다.
  - `lib/supabase.js` 파일을 생성하여 Vercel 마켓플레이스에서 자동으로 설정된 `SUPABASE_URL` 및 `SUPABASE_ANON_KEY` 환경 변수를 사용하는 싱글톤 클라이언트를 구축했습니다.
  - 이제 백엔드 API에서 Supabase 데이터베이스 및 인증 기능을 즉시 사용할 수 있습니다.
## [2026-04-24 13:04 KST]
### ✨ 새로 추가된 기능 (Added)
- **나의 일기 히스토리 기능 구현**:
  - 하단에 과거 일기 내역을 한눈에 볼 수 있는 '나의 일기 히스토리' 섹션을 추가했습니다.
  - `/api/history` 엔드포인트를 통해 Redis에 저장된 모든 기록을 최신순으로 가져옵니다.
  - 각 기록은 프리미엄 글래스모피즘 카드 스타일로 렌더링되며, 원본 일기와 AI 답변을 모두 포함합니다.
  - 일기 분석 성공 시 목록이 실시간으로 갱신되도록 자동 로딩 로직을 추가했습니다.
## [2026-04-24 12:43 KST]
### 🔄 변경된 사항 (Changed)
- **Redis 라이브러리 교체 (`ioredis`)**: 제공해주신 `redis://` 방식의 TCP 연결 문자열을 지원하기 위해 기존 `@upstash/redis`를 삭제하고 `ioredis` 라이브러리로 교체했습니다. 이제 서버리스 함수 외부에서 클라이언트를 선언하여 연결을 재사용합니다.
- **환경 변수 로컬 반영**: `.env` 파일에 실시간으로 제공해주신 `REDIS_URL`을 안전하게 추가하여 로컬 개발 환경에서도 즉시 사용할 수 있도록 설정했습니다.
## [2026-04-24 12:33 KST]
### ✨ 새로 추가된 기능 (Added)
- **Vercel Serverless Redis (Upstash) 연동**:
  - 분석이 성공적으로 완료될 때마다 사용자의 일기와 AI의 답변을 Redis 데이터베이스에 영구 저장합니다.
  - 키 형식: `diary-YYYYMMDDHHMMSS` (KST 기준 고유 ID).
  - `@upstash/redis` SDK를 도입하여 서버리스 환경에 최적화된 REST 방식 통신을 구현했습니다.
## [2026-04-24 12:08 KST]
### 🔄 변경된 사항 (Changed)
- **API 사용량(Quota) 제한 대응**: 구글 무료 등급 계정에서 `gemini-2.0-flash` 모델의 호출 한도가 0으로 설정되어 발생하는 에러(`Quota exceeded... limit: 0`)를 우회하기 위해, 무료 제공량이 확실한 `gemini-2.5-flash` 모델로 다시 롤백했습니다.
## [2026-04-24 11:57 KST]
### 🐛 버그 수정 (Fixed)
- **최신 AI 모델로 마이그레이션**: 구글 API에서 `gemini-1.5-flash` 모델이 단종(Deprecated) 및 삭제됨에 따라, 백엔드의 호출 모델을 현재 서비스 중인 최신 안정화 버전인 `gemini-2.5-flash`로 전면 교체하여 500 API 에러를 완벽하게 해결했습니다.
## [2026-04-24 11:49 KST]
### 🐛 버그 수정 (Fixed)
- **Gemini API 버전 오류 수정**: Vercel 백엔드 함수(`api/analyze.js`)에서 Gemini 1.5 Flash 모델 호출 시 호환되지 않는 `v1` 엔드포인트를 사용하여 발생하던 500 에러(Not Supported)를 해결하기 위해 `v1beta` 엔드포인트로 롤백 및 수정 완료.

## [2026-04-24 11:45 KST]
### 🔄 변경된 사항 (Changed)
- **백엔드 에러 로깅 강화**: 브라우저에서 서버 에러 발생 시, "서버 내부에서 오류가 발생했습니다"라는 모호한 문구 대신 실제 에러 원인(`error.message`)을 상세히 출력하도록 `api/analyze.js`의 catch 블록 수정.

## [2026-04-24 10:50 KST]
### ✨ 새로 추가된 기능 (Added)
- **백엔드(Vercel Serverless)로 API 로직 완벽 이전**:
  - `api/analyze.js`에 실제 Gemini API 호출 로직을 구현하여 프론트엔드에서 백엔드로 분석 역할을 위임.
  - 프론트엔드의 `script.js`는 이제 외부 API가 아닌 자체 서버(`/api/analyze`)와 통신.
- **보안성 극대화 (API 키 은닉)**:
  - 클라이언트 사이드에서 사용하던 `env.js` 파일을 삭제하고, API 키를 `.env`에 `GEMINI_API_KEY`로 설정하여 오직 서버의 환경 변수(`process.env.GEMINI_API_KEY`)로만 접근하도록 변경 (키 유출 위험 차단).

## [2026-04-24 10:38 KST]
### ✨ 새로 추가된 기능 (Added)
- **로컬 스토리지 (Local Storage) 연동**: 
  - AI 분석이 완료되면 작성한 일기 내용과 AI 답변이 브라우저 로컬 스토리지에 자동 저장됨.
  - 페이지 새로고침 시 저장된 데이터를 불러와 입력창과 답변 박스를 이전 상태로 자동 복구함.

## [2026-04-24 10:00 KST]
### 🔄 변경된 사항 (Changed)
- **Gemini API 연동 고도화**:
  - 사용자 환경(Live Server)과의 호환성을 극대화하기 위해 환경 변수 관리 방식을 `env.js` 기반으로 변경 (당시 기준).
  - API 프롬프트를 개선하여, AI가 반드시 '감정: [요약]' 형식과 줄바꿈을 엄격하게 지켜 답변하도록 수정.

## [2026-04-24 09:00 KST] - 초기 개발 단계
### ✨ 새로 추가된 기능 (Added)
- **기본 레이아웃 및 UI 구성**:
  - '오늘 하루는 어땠나요?' 제목 및 여러 줄 텍스트 입력창(`textarea`) 추가.
  - 음성 인식 및 분석 요청 버튼 추가. (Lucide 아이콘 적용)
  - 글래스모피즘(Glassmorphism)과 그라데이션을 적용한 프리미엄 UI 디자인 적용 (`index.css`).
- **음성 인식 (Voice Input) 기능**:
  - Web Speech API를 활용하여 사용자의 음성을 텍스트로 변환 후 입력창에 추가하는 기능 구현.

- **Gemini API 최초 연동**:
  - 작성된 일기 내용을 분석하여 심리 상담사 페르소나의 위로 및 응원 메시지를 제공하는 로직 최초 구현.
